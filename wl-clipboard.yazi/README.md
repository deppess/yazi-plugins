# wl-clipboard.yazi

Wayland implementation of a simple system clipboard for Yazi.

Requires [`wl-clipboard`](https://github.com/bugaevc/wl-clipboard) (make sure `wl-copy`/`wl-paste` are on your `$PATH`).

Forked from [grappas/wl-clipboard.yazi](https://github.com/grappas/wl-clipboard.yazi), itself forked from [orhnk/system-clipboard.yazi](https://github.com/orhnk/system-clipboard.yazi). Fixed and tuned to work on my setup (Arch, niri, Wayland).

## Installation

```sh
ya pkg add deppess/yazi-plugins:wl-clipboard
```
